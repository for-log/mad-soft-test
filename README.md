# Тестовое для MadSoft (Python-разработчик)
# MEMs =)

## Стек
- Nginx
- Docker
- FastApi
- Minio (S3)
- Postgresql

## Функционал (см. http://localhost/docs)
- Добавление мема с текстом
- Обновление мема
- Удаление мема
- Просмотр мемов

## Запуск
1. Установка Docker (https://docs.docker.com/get-docker/)
2. ```bash
   # В папке с проектом
   docker compose up
    ```
3. http://localhost/docs
